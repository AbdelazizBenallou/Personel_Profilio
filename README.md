# Personal Portfolio Website

A modern and responsive personal portfolio website built with HTML, CSS, and JavaScript. Showcases skills, experience, projects, and contact information.

## Features

- About Me section with services overview
- Resume & Experience timeline with skill bars
- Filterable project portfolio (Web Design, Web Development, Applications, Networks Labs, Project)
- Blog section
- Contact form with Google Maps integration
- CV download button
- Fully responsive design (5 breakpoints)
- Dark theme with blue accent gradient
- Custom scrollbar and smooth animations

## Built With

- **HTML5** - Structure
- **CSS3** - Styling & responsive design
- **JavaScript** - Interactivity (SPA navigation, filtering, form validation)
- **Ionicons** - Icons
- **Google Fonts** - Poppins font

## Project Structure

```
portfolio/

└── README.md
```

## Usage

Simply open `index.html` in your browser.

## Customization

- Edit **index.html** for content
- Modify **assets/css/style.css** for design (CSS variables in `:root`)
- Update **assets/js/script.js** for interactivity
- Replace images in `assets/images/`
- Update social links and contact info in `index.html`
